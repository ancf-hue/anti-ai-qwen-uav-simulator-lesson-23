## 📖 О проекте

**«Выживший»** — это интерактивный 3D-симулятор, разработанный в рамках бесплатной социальной офлайн-программы **Anti-AI Qwen Academy**. Тренажёр моделирует работу оператора БПЛА в зоне поражения, где игрок должен выживать против нарастающих волн вражеских дронов.

Проект направлен на развитие:
- 🎯 Пространственного мышления и реакции
- 🧠 Тактического планирования в условиях стресса
- 🕹️ Навыков управления беспилотными системами
- ⚡ Эргономики и цифровой гигиены оператора

> ⚠️ **Важно:** Проект полностью **офлайн-совместим** — работает без интернета после первой загрузки.

## ✨ Особенности

### 🌊 Система волн
- Нарастающая сложность: каждая волна приносит больше врагов
- Враги появляются со всех сторон по радиальной траектории
- ИИ-противников с адаптивным поведением (преследование цели)

### 🎚️ Три уровня сложности

| Уровень | Волн | Врагов/волна | Скорость врагов | Здоровье |
|---------|------|--------------|-----------------|----------|
| 🟢 **Лёгкий** | 5 | 3 | Низкая | 150 HP |
| 🟡 **Средний** | 8 | 5 | Средняя | 100 HP |
| 🔴 **Сложный** | 10 | 7 | Высокая | 70 HP |

### 🚁 Детализированная 3D-модель дрона
- 4 мотора с вращающимися трёхлопастными винтами
- Карбоновая рама, титановые стойки, позолоченные колокольчики моторов
- Подвесная камера с объективом и системой стабилизации
- Навигационные LED (зелёные/красные)
- Динамическое освещение (PointLight)
- Реалистичный наклон корпуса при манёврах

### 🎨 Визуальные эффекты
- Анимированные градиентные рамки (CSS)
- Пульсирующий логотип с drop-shadow
- Динамический туман (`FogExp2`)
- Мягкие тени (`PCFSoftShadowMap`)
- ACES Filmic тонмаппинг
- Анимации свечения кнопок

### 🔊 Адаптивный звук
- Фоновая музыка (loop)
- Процедурно генерируемые SFX через **Web Audio API**:
  - Взрывы, столкновения, предупреждения
  - Мелодия победы (арпеджио C-E-G-C)
  - Звук начала волны

## 🎛️ Управление

| Клавиша | Действие |
|---------|----------|
| `W` / `↑` | Движение вперёд (по курсу камеры) |
| `S` / `↓` | Движение назад |
| `A` / `←` | Strafe влево |
| `D` / `→` | Strafe вправо |
| `Q` / `Shift` | Набор высоты |
| `E` / `Ctrl` | Снижение |
| `ЛКМ` + движение мыши | Обзор 360° (орбитальная камера) |

### 🎯 Механика боя
- **Столкновение с врагом** (< 2 ед.) → враг уничтожен, +1 к счётчику
- **Сближение** (< 3 ед.) → получение урона (8 HP каждые 500 мс)
- **HP = 0** → поражение
